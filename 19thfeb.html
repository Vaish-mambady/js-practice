Javascript Let:

Block Scope:

Global Scope:

variables declaews as var(always global, no block) declared outside {} can be accessed anywhere

Cannot be Redeclared:

Redeclaring Variables:

var can be redeclared, if inside a block, will also redeclare outside the block.

Difference between var, let and const:

var: no scope,can redeclare, can reassign
let: has block scope, cant be redeclared, can be reassigned
const: has block scope, cant be redeclared, cant be reassigned

Redeclaring:

var can be redeclared.
